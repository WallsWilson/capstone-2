


# Body Shop Website

Welcome to the Body Shop Website! This website serves as an online platform for our body shop business, allowing customers to explore our services, schedule appointments, read customer reviews, and contact our team. I have deployed the website at https://jordanbodyshop.onrender.com

## Features

The Body Shop Website offers the following features:

### Homepage

- The homepage provides an overview of our body shop services, showcasing our expertise and highlighting any special offers or promotions.

### Employee Contact

- The website includes a contact section where customers can find the contact information for all employees. This allows customers to reach out to specific team members with any questions or inquiries.

### Reviews

- Our website incorporates a reviews feature, which enables customers to view feedback and testimonials from previous clients. These reviews provide valuable insights into the quality of our services and help potential customers make informed decisions.

### Appointment Scheduling

- The website integrates with the Timekit API to provide a seamless appointment scheduling experience. Customers can select their preferred date and time for an appointment, and the system will handle the scheduling process, ensuring smooth coordination between customers and our body shop team.

## Technologies Used

The Body Shop Website is built using the following technologies:

- Python: The backend of the website, including the reviews feature, is developed using the Python programming language.
- Flask: The Flask framework is used to create the web application and handle routing and request handling.
- Timekit API: Integration with the Timekit API allows for efficient and accurate appointment scheduling. https://admin.timekit.io/a/apps/test-jordan-walls-wilson-8932/resources
- Bootstrap: This is what styles the full app.

## Installation and Setup

To run the Body Shop Website locally, follow these steps:

1. Clone the repository: `git clone [repository URL]`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Set up the configuration: Update the necessary configuration files (if any) to provide API keys, database connections, or other required parameters.
4. Start the web server: `flask run`
5. Access the website: Open a web browser and navigate to `http://localhost:5000` to view the website locally.
